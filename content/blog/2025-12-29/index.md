The project directory for my personal site is a mess and i’m trying to restructure and reorganize everything from scratch. Earlier, i used `hugo-xmin` theme created by [Yihui Xie](https://yihui.org/). However as i try to customize this theme, due to me being dumb, i break everything. Now, learning from that mistake i feel like i don’t want any breaking to be catasthrophic in the future where i already have a lot of blog posts (maybe i'm too paranoid...). For that reason, i’m ditching the `hugo-xmin` theme and create everything from scratch so i understand the technology deep enough to not break everything catasthrophically in the future.

Since this site was built from scratch, it's incredibly minimal and lack any
advanced features; it's just `HTML` and plain vanilla `CSS`. I like minimalist design philosophy. However, i'm still a normal human who want shiny things; there are still things that i want to implement: 
- LaTeX rendering
- Table of Content
- Dark Mode 
- (maybe) Tag System Navigation

Well, since this is my first time building website and i’m not particularly fluent in any web related technologies, i consider my goal to have the personal site to be done because this site, though incredibly simple, is already capable of doing most of the things that i wanted to do. The above improvements and features addition will be carried out later, most likely next years, when i really have the free time to work on it.
